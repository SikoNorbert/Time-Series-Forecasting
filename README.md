# Time Series Forecasting
## Chips: AMD, ASUS, INTEL, MSI, NVIDIA

### Setup:
1. Download the repository to local
2. Install **PyCharm** from JetBrains
3. Setup project with project files requirement by using the command: <ins> pip install -r requirements.txt </ins>
4. Run the following command for Web GUI: <ins>streamlit run main.py</ins>

#### You will see that the program will run only for PT (PyTorch) and not TF (TensorFlow).
